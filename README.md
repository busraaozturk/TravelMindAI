# TravelMind AI

Bütçenize, sürenize, ilgi alanlarınıza ve seyahat tarzınıza göre saat saat planlanmış kişisel seyahat rotaları oluşturan, yapay zeka destekli seyahat danışmanı.

## İçerik

- `travelmind-ai.html` — Tek dosyalık web uygulaması (HTML/CSS/JS).
- `Prompt.docx`, `PromptDuzenlemesi.pages` — Proje notları ve prompt dokümanları.

## Özellikler

- 4 adımlık sihirbaz ile kişiselleştirilmiş seyahat planı oluşturma
- Bütçe dağılımı ve günlük rota planlaması
- Kayıtlı seyahatler
- Formspree entegreli iletişim formu
